# SpringBoot

A collection of Spring Boot sample projects and learning exercises.

## Repository Structure

This workspace contains multiple Spring Boot and related Java projects, including examples for:

- Spring Boot application development
- Spring Data JPA
- REST APIs
- Reactive programming and RSocket
- Spring Batch and JMS
- Thymeleaf and web applications
- MongoDB and file processing

## How to Use

Each project folder is a separate Maven project with its own `pom.xml`.

To build a project, open a terminal in the desired project folder and run:

```powershell
./mvnw clean package
```

To run a Spring Boot application, use:

```powershell
./mvnw spring-boot:run
```

## Notes

- The root workspace is intended for learning and experimentation.
- Projects may have different dependencies and runtime requirements.
- Inspect each project's `HELP.md` for any specific instructions.
